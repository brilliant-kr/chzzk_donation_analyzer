# 치지직 후원 분석기
치지직(Chzzk) 후원 데이터를 분석하고 시각화하는 도구입니다.  
[Windows 다운로드](https://github.com/brilliant-kr/chzzk_donation_analyzer/releases/download/v3-windows/chzzk_donation_analyzer.exe)


## 기능
- 채널별 후원 금액 집계
- 연도/월/요일별 통계
- 최대 단건 후원 정보
- 다양한 그래프 시각화
- GUI 인터페이스
- 치지직 API에서 직접 데이터 가져오기

## 분석 항목
- **요약**: 최대 단건 후원, 전체 통계, 연도별 합계
- **채널별**: 채널별 후원 금액 순위
- **시간별**: 월별/요일별 후원 통계
- **그래프**:
  - 채널별 TOP 10
  - 연도별 후원 금액
  - 월별 후원 추이
  - 요일별 평균 후원 금액
  - 채널별 비율 (파이 차트)
  - 월별 후원 건수
